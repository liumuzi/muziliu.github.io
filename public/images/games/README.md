# 游戏项目图片目录

## 图片命名规范

每个游戏项目使用其 `id` 作为图片文件名：

- `dating-show.jpg` - 我的恋综超失控
- `bug-decor.jpg` - 虫虫家装
- `not-again-hero.jpg` - Not Again, Hero
- `error-guide.jpg` - 报错漫游指南
- `farty-party.jpg` - Farty Party

## 图片规格建议

- **格式**：JPG 或 PNG（JPG 推荐，文件更小）
- **尺寸**：建议 1200x675 像素（16:9 比例）
- **文件大小**：单张不超过 500KB
- **内容**：游戏截图、概念图或主视觉

## 使用方法

将图片文件放入此目录后，它们会自动显示在游戏卡片上。

图片会被引用为：`/images/games/{game-id}.jpg`
